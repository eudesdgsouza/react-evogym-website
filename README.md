# Gym App 🏋️ 

Aplicativo de ginástica desenvolvido com foco em uma interface moderna e responsiva, utilizando React, TypeScript e TailwindCSS.


## 🛠️ Tecnologias Utilizadas

- **React** - Biblioteca para construção de interfaces interativas
- **TypeScript** - Tipagem estática para segurança e previsibilidade no código
- **TailwindCSS** - Framework para estilização rápida e responsiva

## 🚀 Funcionalidades

- Interface de usuário intuitiva para gestão de atividades de ginástica
- Estilização com TailwindCSS para um design limpo e responsivo

## 📈 Aprendizado

Este projeto proporcionou:

1. Configuração de TypeScript em um projeto React com Vite
2. Configuração e uso de TailwindCSS em projetos React

## ⚙️ Instalação e Execução

Siga os passos abaixo para rodar o projeto localmente:

1. Clone o repositório:
   ```bash
   git clone https://github.com/eudesdgsouza/react-evogym-website.git
   ```

2. Navegue até o diretório do projeto:
   ```bash
   cd react-evogym-website
   ```

3. Instale as dependências:
   ```bash
   npm install
   ```

4. Inicie o projeto:
   ```bash
   npm run dev
   ```

5. Abra seu navegador em [http://localhost:3000](http://localhost:3000) para visualizar o aplicativo.
